    This method works in a similar manner to ‘Files#move move’ method when the ‘StandardCopyOption#ATOMIC_MOVE ATOMIC_MOVE’ option is specified. That is, this method moves a file as an atomic file system operation. If the ‘srcpath’ parameter is an ‘Path#isAbsolute absolute’ path then it locates the source file. If the parameter is a relative path then it is located relative to this open directory.
