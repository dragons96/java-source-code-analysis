    classEnabled: boolean[] null default 否 否 否 否 否 A parallel array to classes, indicating whether each class is to have assertions enabled or disabled. A value of true for classEnabled[i] indicates that the class named by classes[i] should have assertions enabled; a value of false indicates that it should have classes disabled.
