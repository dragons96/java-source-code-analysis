    The TextArea class defines several constants that can be supplied as values for the scrollbars argument:

    • SCROLLBARS_BOTH,
    • SCROLLBARS_VERTICAL_ONLY,
    • SCROLLBARS_HORIZONTAL_ONLY,
    • SCROLLBARS_NONE.
    Any other value for the scrollbars argument is invalid and will result in this text area being created with scrollbar visibility equal to the default value of ‘#SCROLLBARS_BOTH’.    		 HeadlessException
