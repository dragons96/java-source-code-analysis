    • ‘HH:mm:ssXXXXX’
    • ‘HH:mm:ss.SSSXXXXX’
    • ‘HH:mm:ss.SSSSSSXXXXX’
    • ‘HH:mm:ss.SSSSSSSSSXXXXX’
    The format used will be the shortest that outputs the full value of the time where the omitted parts are implied to be zero.
