    connect(PipedWriter src): void public 否 否 否 否 否 否 Causes this piped reader to be connected to the piped writer src. If this object is already connected to some other piped writer, an IOException is thrown.

    If src is an unconnected piped writer and snk is an unconnected piped reader, they may be connected by either the call: 

    snk.connect(src)

    or the call: 

    src.connect(snk)

    The two calls have the same effect.
