    serialVersionOnStream: int currentSerialVersion private 否 否 否 否 否 Describes the version of DecimalFormatSymbols present on the stream. Possible values are:
    • 0 (or uninitialized): versions prior to JDK 1.1.6.
    • 1: Versions written by JDK 1.1.6 or later, which include two new fields: monetarySeparator and exponential.
