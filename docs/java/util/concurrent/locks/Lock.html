    unlock(): void public 否 否 否 否 否 是 释放锁

    实现注意事项

    一个“锁”实现通常会对哪个线程可以释放锁施加限制（通常只有锁的持有者可以释放锁），如果违反了该限制，可能会抛出（未经检查的）异常。任何限制和异常类型必须由该“锁定”实现记录。
