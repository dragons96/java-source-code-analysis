    getInstance(Locale locale): Currency public 是 否 否 否 否 否 Returns the Currency instance for the country of the given locale. The language and variant components of the locale are ignored. The result may vary over time, as countries change their currencies. For example, for the original member countries of the European Monetary Union, the method returns the old national currencies until December 31, 2001, and the Euro from January 1, 2002, local time of the respective countries.

    The method returns null for territories that don’t have a currency, such as Antarctica.
